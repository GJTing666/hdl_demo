  • 番茄火锅

    番茄火锅

    一定要喝汤的番茄火锅

    精选日照充裕的阳光番茄，制酱炒制而成的番茄底料；推荐搭配河鲜类食材。

    覆盖区域 : 全国覆盖

  • 牛油麻辣火锅

    牛油麻辣火锅

    慢火熬煮4小时的牛油锅底

    选用牛油与茂汶等原产花椒，越煮越香，不变味，搭配蒜泥香油味碟，这样吃才地道。

    覆盖区域 : 全国覆盖

  • 菌汤火锅

    菌汤火锅

    菌菇慢火熬煮2小时

    锅底汤色黄褐，清澈；菌香浓郁、爽口；推荐搭配涮食菌菇类食材。

    覆盖区域 : 全国覆盖

  • 清油麻辣火锅

    清油麻辣火锅

    植物油炒制，香辣浓郁不油腻

    严选花椒、辣椒、香辛料原料，地道川味火锅；传统工艺发酵豆瓣；汤底色泽鲜红，口感香辣浓郁；推荐搭配涮食脆弹类动物食材。

    覆盖区域 : 全国覆盖

  • 三鲜火锅

    三鲜火锅

    色泽纯正，口感鲜香浓郁

    选用现代工艺萃取的结合肉类浓缩骨汤为汤底；可根据喜好搭配不同的特色味碟均可， 配合芝麻酱更佳；推荐搭配涮食海鲜类及素食类食材。

    覆盖区域 : 全国覆盖

  • 猪肚鸡火锅

    猪肚鸡火锅

    汤头乳白、鲜香浓郁

    精选猪肚、三黄鸡和慢火熬制的风味猪肚汤料包搭配，鲜香浓郁、椒香扑鼻；推荐搭配涮素食食材。

    覆盖区域 : 江苏省、上海、浙江省、福建省、湖北省、湖南省

  • 猪肚乌鸡锅

    猪肚乌鸡锅

    Q弹猪肚，滑脆乌鸡卷，鲜香够味

    Q弹猪肚，滑脆乌鸡卷，慢火熬制的浓郁猪骨汤底，鲜香够味； 推荐搭配弹脆类动物食材及素食食材。

    覆盖区域 : 上海

  • 椒香火锅

    椒香火锅

    萃取辣椒、花椒香味、椒香浓郁

    高温萃取和慢火熬制而成； 推荐搭配涮食脆弹类动物食材。

    覆盖区域 : 江苏省、山东省、安徽省、河北省、云南省、贵州省

  • 辣炫风火锅

    辣炫风火锅

    一款嗨辣的火锅

    辣味升级，青红鲜椒、花椒油大火翻炒而成； 鲜椒自然浓郁、口味鲜辣清爽，推荐搭配涮食脆弹类动物食材。

    覆盖区域 : 北京、河南省、山东省、湖北省、湖南省、河北省、天津、江西省、贵州省

  • 牛肉汤火锅

    牛肉汤火锅

    有牛肉块的特色火锅

    通过慢火熬煮，汤味醇正，肉香浓郁， 咸鲜而不腻；推荐搭配涮素食食材。

    覆盖区域 : 江苏省、广东省、上海、浙江省、福建省、云南省、海南省

  • 青椒火锅

    青椒火锅

    青辣椒与青花椒炒制，鲜辣爽口

    精选青辣椒、青花椒大火熬制2小时； 只用植物油，好吃不油腻，推荐搭配涮食脆弹类动物食材。

    覆盖区域 : 江苏省、广东省、陕西省、河南省、 山东省、福建省、河北省、云南省、四川省、广西省、甘肃省、海南省、宁夏省、贵州省

  • 泰式冬阴功

    泰式冬阴功

    沸腾鸭掌，巴适得很！

    冬阴功，“冬阴”是酸辣的意思， “功”是虾的意思，是泰国的一道富有特色的酸辣口味汤品，在泰国非常普遍。推荐搭配海鲜类物食材。

    覆盖区域 : 江苏省、广东省、北京、上海、山东省、浙江省、福建省、湖北省、安徽省

  • 鸭掌火锅

    鸭掌火锅

    沸腾鸭掌，巴适得很！

    卤制鸭掌配以辣油锅底；汤底色泽鲜红， 口感香辣浓郁；推荐搭配涮食脆弹类动物食材。

    覆盖区域 : 江苏省
